## рџЊџ About The Project

**Smart AI Pathfinding Visualizer** is an interactive Artificial Intelligence project that demonstrates how intelligent search and pathfinding algorithms work in real-time through dynamic grid-based visual simulations.

The application allows users to create obstacles, define start and destination nodes, and watch AI algorithms intelligently discover the shortest path step-by-step.

This project was developed as part of the **Fundamentals of Artificial Intelligence** course to better understand graph traversal, heuristic optimization, and intelligent decision-making techniques.
